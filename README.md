# Dotfiles

This repository contains my configuration files for various applications.

## Available Configurations

- Alacritty
- Fish
- Ghostty
- Kitty
- Neovim
- Starship
- Tmux
- i3
- Polybar

### Install with stow

```sh
stow .
```

### Install TPM

```sh
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
```
